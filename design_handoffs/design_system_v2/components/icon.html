Decision rules

  • UseAlways via <EdIcon name="search" />. Never import lucide-react directly into a feature component.
  • UseSemantic color prop (brand / success / warning / danger). Avoid inline color overrides.
  • PreferStroke icons (default). Solid variant only when filled affordance is meaningful (selected pin, completed step).
  • AvoidMulti-color icons. Single-color, stroke-based icons keep the visual rhythm.
  • AvoidEmoji as a substitute for an icon.

A11y

  • RequiredDecorative icons (inside labeled buttons, tags, etc.): aria-hidden="true".
  • RequiredStandalone icon with meaning: aria-label or wrap in EdIconButton (which requires it).
  • RequiredStroke icons stay legible at 12px because line-weight scales with size.

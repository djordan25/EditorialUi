Decision rules

  • Use3–10 rows of static metadata. Side-panel detail blocks. Modal summaries before / after sections.
  • PreferEdDataTable when there's any sort, selection, filter, or pagination.
  • PreferA <dl>-based "key-value list" component if you'd never want columns — but EdNativeTable handles both with one primitive.
  • AvoidHand-rolling tables. Even static blocks inherit type, spacing, and dark-mode tokens through this primitive.

A11y

  • RequiredFor key-value: <th scope="row"> on label cells. Screen readers announce "ID: F-2438".
  • RequiredFor columned: <th scope="col">. Add <caption> when the table title isn't visible above.
  • RequiredSame dark-mode contrast rules as EdDataTable — uppercase mono labels stay at AA.

Layout

  • StackVertical: label → control → hint. Gaps are 6px (label→control) and 6px (control→hint).
  • InlineFor switches and checkboxes inside settings rows, label is on the left and switch is flush right. The component handles both modes via a layout prop.
  • PreferEdTextField over EdFormControlLabel + raw input — the bundled version is what most forms want.

A11y

  • RequiredComponent generates a stable id if the control doesn't have one; wires htmlFor on label + aria-describedby on control.
  • RequiredError text gets role="alert" + aria-live="polite" when error transitions on.

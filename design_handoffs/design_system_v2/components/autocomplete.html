credit-risk retail

Suggestion behavior

  • RequiredShow suggestions only when query length ≥ 2 chars (configurable per use).
  • RequiredHighlight matched substring with <span class="hi">. Use brand bg, not yellow.
  • RequiredFree text always valid — pressing Enter without selecting commits the typed value.
  • Required"Create new" item appears when no exact match exists. Style as a divider-separated action row.
  • AvoidForcing users into existing values. That's EdComboBox.

Decision rules

  • UseFinding titles, tag names, ad-hoc business unit, custom finding category. The value can be anything.
  • PreferEdComboBox when value must be from a known list (owner, severity).
  • PreferEdInput alone when there's nothing useful to suggest.

A11y

  • RequiredSame Radix Combobox base as EdComboBox, with aria-autocomplete="both".
  • RequiredLive region: "{n} suggestions, use arrows to browse" on first character.
  • RequiredHighlighted substring is decorative — screen readers read full label, not the highlight markup.

No findings in this period

Findings appear here once an examination opens. You can open one from the audit period selector.

Visual — small, restrained. Single-glyph icon in dashed circle. No illustrations.
Title — what's missing, in plain language. Sentence case, no punctuation.
Body — why and what unblocks it. ≤ 2 sentences.
Actions — 0, 1, or 2. Primary is the unblocking action; secondary is a side path.

Copy guidance

  • TitlePlain. "No findings yet" not "Nothing to see here!". Avoid exclamation, slang, jokes.
  • BodyState the cause AND the unblock. "Findings appear here once an examination opens. Open one from the period selector."
  • ActionsPrimary verbs only. "Create finding", "Clear filters", "Retry". Skip CTAs the user can't perform.
  • AvoidAnimated illustrations, mascots, "Oops!" copy, or apologies. This is institutional software.

Decision rules

  • UseInside table bodies, list panels, drawer/inspector contents, dashboard widgets when no data is present.
  • PreferEdNotification when content exists but a state needs flagging (banner above the data).
  • PreferSkeletons (EdSkeleton) during initial load — empty state is for genuinely empty.
  • AvoidEmpty state in modals or quickly-resolved areas. Just hide the section.

A11y

  • RequiredTitle is the first heading inside the host region. Use the heading level appropriate to placement.
  • RequiredVisual icon is decorative — aria-hidden="true". Title carries meaning.
  • RequiredButtons inherit standard EdButton a11y.

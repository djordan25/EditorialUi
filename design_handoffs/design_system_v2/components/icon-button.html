Decision rules

  • UseToolbar / TopBar utilities (search, filter, refresh, settings, more).
  • UseInline table-row actions (edit, copy, delete) — bordered variant.
  • UseToggle behavior — set aria-pressed on the button. Selected state uses brand-bg tint.
  • AvoidIcon buttons in dialog footers — use labeled EdButton. Footer space is the most label-needy place in the UI.
  • AvoidStacking 4+ icon buttons without grouping by type — overflow the rest into an EdMenu.

A11y

  • RequiredEvery instance must pass aria-label. Component prop type makes this required.
  • RequiredToggle behavior uses aria-pressed="true|false", not aria-checked.
  • RequiredAlways wrap with EdTooltip when label isn't otherwise visible (which is most of the time).

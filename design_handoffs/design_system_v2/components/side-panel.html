Behavior

  • RequiredSlide in from right edge in 200ms. Slide out matches.
  • RequiredNon-modal: page outside drawer remains keyboard- and click-reachable. Modal: focus trap + overlay.
  • RequiredEsc closes (modal) or dismisses (non-modal, no unsaved changes). Unsaved-changes guard prompts inline.
  • RequiredURL state: drawer open state encoded in ?drawer=<id> so deep linking + back/forward works.
  • AvoidResizable handles. Pick a size that fits the content type and stick with it.

Decision rules

  • UseInspecting a row while keeping the list visible. Detail panel for selected entity. Quick reassignment / labelling that doesn't deserve a modal.
  • PreferEdModal for self-contained, full-attention workflows (new finding, bulk action).
  • PreferFull page when content needs the full canvas (graph builder, monaco editor).

A11y

  • RequiredBuild on Radix Dialog with modal={false} for non-modal; modal={true} for modal mode.
  • RequiredTitle is in DOM as aria-labelledby target. First focusable receives focus on open; on close, focus returns to trigger.
  • RequiredLive region announces "Inspecting finding F-2438" on open.

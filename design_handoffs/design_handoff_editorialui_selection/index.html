Bundle 3 — Selection

Three components for picking from a list: EdSelect (compact, fixed options), EdComboBox (searchable single / multi, with async), and EdAutocomplete (free text + suggestions). Plus a private _internal/ module that the two combobox-shaped components share.

05 What this bundle codifies

  • useListboxNav is the single sourceEvery combobox-shaped component reuses it. Future date pickers + mention menus must too.
  • Options are plain objects{ value, label, meta?, group?, disabled? }. No <Option> children. No loadOptions / cacheOptions zoo.
  • Multi emits arraysEdComboBox multiple emits string[]. The trigger renders selected values as removable, focusable tag chips.
  • Free text is a different componentEdComboBox constrains values to its list. EdAutocomplete doesn't — suggestions are hints, not constraints.
  • Highlight is decorativeBrand-bg substring highlight via .edmenu__hi. Screen readers read the full label, not the wrapped markup.

Decision rules

  • UseWrap mode in side panels, detail views, drawer bodies — anywhere height is cheap.
  • UseSingle-row in table cells. Collapse to "+ N more" with a tooltip showing the rest.
  • AvoidInventing a separate API per consumer. The container handles all gap/overflow logic.

A11y

  • RequiredWrapper is a <div> (no role). Each tag is its own readable unit.
  • RequiredOverflow counter is keyboard-reachable when it opens a popover; otherwise decorative.

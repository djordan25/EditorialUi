Behavior

  • RequiredBuild on Radix Popover. Non-modal — page scroll continues, surrounding content stays interactive.
  • RequiredOutside click dismisses. Esc dismisses and returns focus to trigger.
  • RequiredAuto-flip placement when bubble would clip viewport. Default placement is below (bottom-start).
  • RequiredOpen animation: 120ms fade + 4px slide from anchor.
  • AvoidPopovers that require scrolling. If content needs height, move to a drawer or modal.

Decision rules

  • UseQuick edit of a single value, filter panels, definitions, overflow action menus, simple attribute pickers.
  • PreferEdTooltip for read-only hints (no interactive content).
  • PreferEdMenu for command lists — same look, but role-specific a11y wiring.
  • PreferEdModal for forms with more than a single field, or where confirmation matters.

A11y

  • RequiredTrigger has aria-expanded + aria-controls pointing at popover ID.
  • RequiredPopover has role="dialog" when it contains form controls; omit when it's a passive disclosure.
  • RequiredFocus moves into popover on open; first focusable receives focus.

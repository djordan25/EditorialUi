Decision rules

  • Use≤7 fixed, known options. Severity, cadence, role, environment, status filter.
  • UseGrouped sections for related options (Active / Archived). Use a separator + group label.
  • PreferEdComboBox for >7 options or any list a user might type to find. Switching at 7 isn't a hard line — choose by usage, not count.
  • AvoidFree text. Select is for picking from known options only.
  • Avoid2-option selects — use EdRadioGroup or EdSwitch.

A11y

  • RequiredUse Radix Select. role="combobox" + aria-expanded + aria-controls handled.
  • RequiredType-ahead matches first letter of items. Arrow keys navigate. Enter / Space selects.
  • RequiredSelected item gets aria-selected="true"; visible check icon is decorative.

Decision rules

  • UseDetail pages 2+ levels deep. Nested workflows. Embedded views in modals where context needs to be reasserted.
  • UseCompact back-link variant when the hierarchy is shallow (single parent).
  • PreferThe page title alone when there's no parent. No "Home / Page" — just "Page".
  • AvoidBreadcrumbs as primary navigation. They're orientation, not a route picker.
  • AvoidIncluding the brand or top-level "Home" as a crumb. The TopBar already covers that.

A11y

  • RequiredWrapping <nav aria-label="Breadcrumb"> + <ol> with <li> per crumb.
  • RequiredCurrent crumb sets aria-current="page"; renders as plain text (not a link).
  • RequiredSeparators are aria-hidden="true". Use CSS pseudo-element OR aria-hidden span — never both.
  • RequiredOverflow "…" is a button that opens a menu listing hidden crumbs.

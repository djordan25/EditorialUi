Decision rules

  • UsePicker results, recent items, user / entity selectors, command-palette results.
  • PreferEdDataTable when rows have ≥3 columns, sorting, or selection-with-actions.
  • PreferEdMenu for action lists triggered from a button.
  • PreferEdAccordion when items expand to reveal content.
  • AvoidLists of more than ~12 items without search. Add a filter input above, or switch to a table.

A11y

  • RequiredUse <ul> / <li> when items are static. Selectable list: role="listbox" + role="option" + aria-selected.
  • RequiredUp/Down arrows move focus through items. Enter/Space selects.
  • RequiredGroup labels use role="group" with aria-labelledby.
  • RequiredTrailing meta is included in the accessible name: aria-label="Marcus Chen, 14 open, 2 overdue".

Four segments map to four bands: weak (red, 0-1), fair (orange, 2), good (orange, 3), strong (green, 4). Caption mirrors the band label.

Decision rules

  • UseSign-in, change-password, reset-password screens. Anywhere a password is captured.
  • UseStrength meter on new password only — never on sign-in (leaks information).
  • AvoidReveal-toggle on initial sign-in. The user already knows their password — toggle is shoulder-surf risk for marginal benefit.

A11y

  • RequiredReveal toggle has aria-label "Show password" / "Hide password" — string updates on toggle.
  • RequiredStrength feedback is read by screen readers via aria-live="polite" on the caption only — segment colors are decorative.

Decision rules

  • UsePicking from known lists where users would benefit from typing — owners, regulations, models, model owners, business units.
  • UseMulti mode for tag-style selection. Show selected as EdTag chips inside the trigger.
  • PreferEdAutocomplete when free text is also valid (entering a value not in the list).
  • PreferEdSelect for ≤7 fixed options where a search bar adds friction.
  • AvoidForcing users to scroll long lists with no filter — always show the search input when items > 10.

A11y

  • RequiredBuild on Radix Combobox or HeadlessUI Combobox; uses role="combobox" + aria-autocomplete="list".
  • RequiredSelected item updates aria-activedescendant. Live region announces "{n} results".
  • RequiredEsc closes menu without changing selection. Enter on highlighted item selects.
  • RequiredMulti mode: each selected tag is a focusable chip with its own remove button (keyboard reachable).

Decision rules

  • UseSibling views of the same entity (Overview / Findings / Audit log). Page-level navigation within a route.
  • UseSegmented for inline view toggles (table/inspector/graph). 2–4 short labels only.
  • PreferEdFilterChipRow for filters above a single dataset. Tabs imply different content; filters narrow the same content.
  • PreferEdAccordion when sections are independently expandable rather than mutually exclusive.
  • AvoidMore than 7 tabs. Restructure as a sidebar nav or grouped tabs.
  • AvoidTabs that jump-scroll within one panel. That's a table of contents.

A11y

  • RequiredBuild on Radix Tabs. role="tablist" + role="tab" + role="tabpanel" wired automatically.
  • RequiredLeft/Right arrows move focus between tabs. Home/End jump to first/last. Activation: manual (Enter) by default for async panels.
  • RequiredCount badge included in accessible name: aria-label="Findings, 23".
  • RequiredURL state: active tab encoded in ?tab=findings for shareability.
